## After Creating All Products

### Set up the webhook
1. Go to **Settings → Webhooks**
2. Add endpoint: `https://your-app.vercel.app/api/lemon-webhook`
3. Events: check `order_created`
4. Copy the signing secret → save as `LEMON_WEBHOOK_SECRET` env var

### Note variant IDs
After creating products, note each variant's numeric ID from the URL or API:
```bash
curl -s -H "Authorization: Bearer YOUR_KEY" \
  "https://api.lemonsqueezy.com/v1/variants" | python3 -m json.tool
```

### Test
Create a $0 test product → purchase it → verify the webhook fires and you can send the token email.

## Quick Reference: What Customers Get

| Purchase | Token gives access to |
|----------|----------------------|
| Any single pack | Full `@micrographics-js/react` (or vue/svelte/vanilla) |
| React Bundle | `@micrographics-js/react` + `@micrographics-js/core` |
| Full Library | All 4 framework packages + core |
| Lifetime | Same as Full + future versions |

> **Note:** Since we ship ONE package per framework (not per-pack), all paid customers get the same npm token. The pack/bundle pricing is about perceived value — technically all React customers can access all 84 React components. This is standard for commercial component libraries.
