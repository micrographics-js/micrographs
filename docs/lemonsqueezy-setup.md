## After Creating All Products

1. Go to Settings → Webhooks
2. Add endpoint: your-app.vercel.app/api/lemon-webhook
3. Events: check "order_created"
4. Copy signing secret → LEMON_WEBHOOK_SECRET env var

## Customer Token (Classic PAT)

1. https://github.com/settings/tokens
2. "Generate new token (classic)" (NOT fine-grained)
3. Check only: read:packages
4. No expiration
5. This token goes in the purchase confirmation email
